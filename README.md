# Daniel Rodrigues

This portifolio were created with Vue for study purpuses

## Running

### Client

Enter the client folder

```sh
$ cd client
```

Install dependencies

```sh
$ npm i
```

Run in dev mode

```sh
$ npm run dev
```

### Server

Enter the server folder

```sh
$ cd server
```

Install dependencies

```sh
$ npm i
```

Install nodemon

```sh
$ npm i --global nodemon
```

Run in dev mode

```sh
$ npm run dev
```
